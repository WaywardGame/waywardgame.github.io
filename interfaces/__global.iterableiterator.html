    • Defined in utilities/iterable/prototype/Flat.ts:29

    Returns a new iterator iterating over every value of the result of calling the mapper function on each value in the current iterator.

    For example:

    [[1, 2, 3], [4, 5, 6]]
    .flatMap(numberArray => numberArray
        .map(num => num + 1))
